## Run Locally

Clone the project

```bash
  git clone https://github.com/its-joseph-230606/LlamaGPT.git
```

Go to the project directory
```bash
  cd LlamaGPT
```
```bash
  pip install -r requirements.txt
```

Install dependencies

```bash
  python chat.py
```
